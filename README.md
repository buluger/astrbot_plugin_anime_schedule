# 番剧追番表

AstrBot 插件，用于在群内按周一至周日记录追番列表，支持图片 + 文字上传，生成周表长图，以及每日定时推送今日番剧。

## 功能概览

- 按星期（周一至周日）管理番剧，每条记录包含名称、封面图与观看进度
- 一键生成每周追番长图（`番剧图`）
- 查看今日或指定星期的番剧列表（含已看/应看集数）
- 指令或推送消息表情回应，快速更新进度（群内共享）
- 群内可配置四级操作权限、每日添加上限
- 支持每日定时推送今日番剧到群聊

## 环境要求

- AstrBot `>= 4.16, < 5`
- 支持平台：aiocqhttp、QQ 官方机器人、Telegram

## 安装

1. 将本插件放入 AstrBot 的 `plugins` 目录，或通过插件市场安装
2. 安装依赖：

```bash
pip install -r requirements.txt
```

主要依赖：`Pillow`（生成周表长图）、`aiohttp`（下载图片）

3. 重启 AstrBot 使插件生效

## 插件配置

在 AstrBot 插件配置中可设置以下全局默认值（各群可在群内指令中单独覆盖）：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `perm_mode_default` | 默认操作权限：`1` 仅 Bot 管理员，`2` 仅群主，`3` 群主+管理员，`4` 全员 | `3` |
| `max_per_day_default` | 每天默认可添加番剧数量上限（1~20） | `10` |
| `push_hour_default` | 默认定时推送小时（0~23） | `8` |
| `push_minute_default` | 默认定时推送分钟（0~59） | `0` |

## 指令一览

所有指令均在**群聊**中使用。部分指令支持 `/` 前缀（如 `/番剧帮助`）。

### 帮助

| 指令 | 说明 |
|------|------|
| `番剧帮助` | 显示指令帮助（别名：`/番剧帮助`、`番剧菜单`） |

### 上传与查看

| 指令 | 说明 |
|------|------|
| `番剧上传 周X 名称 [总集数]` + 图片 | 添加番剧；未填总集数时自动查 Bangumi，失败则默认 12 |
| `番剧图` | 生成每周追番长图（含进度） |
| `今日番剧` | 查看今天更新的番剧（文字 + 单行拼接长图） |
| `番剧 周X` | 查看指定星期的番剧（文字 + 单行拼接长图） |

**星期写法支持：** `周一`~`周日`、`周1`~`周7`、`星期一`~`星期日` 等。

**上传示例：**

```
番剧上传 周一 葬送的芙莉莲 12
（同时发送一张封面图，或引用一条带图的消息）
```

### 观看进度

进度为**群内共享**。语义：

- **已看**：已经看了多少集（`watched_ep`）
- **应看**：已看 + 1（未完结时）

| 指令 | 说明 | 权限 |
|------|------|------|
| `已看 周X` | 该天全部番剧各标记看过一集 | 按「番剧权限」 |
| `已看 周X 编号` | 标记指定番剧看过一集 | 按「番剧权限」 |
| `已看 编号` | 默认操作今天的列表 | 按「番剧权限」 |
| `已看全部` | 今日列表全部标记看过 | 按「番剧权限」 |
| `撤回 周X` | 该天全部番剧各撤回一集 | 按「番剧权限」 |
| `撤回 周X 编号` / `撤回 编号` | 撤回指定番剧最近一次已看 | 按「番剧权限」 |
| `番剧已看 周X N` | 该天全部设为已看 N 集 | 按「番剧权限」 |
| `番剧已看 周X 编号 N` | 直接设置指定番剧已看集数 | 按「番剧权限」 |
| `番剧上限 周X` | 该天全部从 Bangumi 同步总集数 | 按「番剧权限」 |
| `番剧上限 周X 编号` | 指定番剧从 Bangumi 同步 | 按「番剧权限」 |
| `番剧上限 周X 编号 N` | 手动设置指定番剧总集数 | 按「番剧权限」 |
| `番剧上限 周X 全部 N` | 该天全部手动设为 N 集 | 按「番剧权限」 |

**示例：**

```
已看 周一
已看 1
已看 周一 1 2
已看全部
撤回 周一
撤回 1
番剧已看 周一 5
番剧已看 周一 1 5
番剧上限 周一
番剧上限 周一 1 24
番剧上限 周一 全部 12
```

只写「周X」时，对该天**全部**番剧执行相同操作；加上编号则只操作一部。

上传时若不写总集数，会自动按番名查询 Bangumi；查不到则回退默认 12 集。也可手动指定：`番剧上传 周一 芙莉莲 12`。

**推送消息表情回应（aiocqhttp / NapCat）：**

对当日推送消息贴表情即可更新进度：

- 数字表情 `1`~`10` / `1️⃣`~`🔟` → 对应编号标记看过
- `👍` / `OK` / `强` / `赞` 等 → 今日全部标记看过

### 删除、移动、交换与清空

| 指令 | 说明 | 权限 |
|------|------|------|
| `删除番剧 周X 编号` | 删除某天指定番剧（编号从 1 开始） | 按「番剧权限」 |
| `移动番剧 周X 编号 周Y` | 将某天指定番剧移到另一天 | 按「番剧权限」 |
| `交换番剧 周X 编号A 周Y 编号B` | 交换两部番剧的位置（可跨天或同天调序） | 按「番剧权限」 |
| `清空番剧 周X` | 清空某天的全部番剧 | 按「番剧权限」 |
| `清空番剧 全部` | 清空本周全部番剧 | 按「番剧权限」 |

**示例：**

```
删除番剧 周一 1
移动番剧 周一 1 周三
交换番剧 周一 1 周三 2
交换番剧 周五 1 周五 2
清空番剧 周三
清空番剧 全部
```

### 权限与设置

| 指令 | 说明 | 权限 |
|------|------|------|
| `番剧权限` | 查看当前操作权限 | 所有人 |
| `番剧权限1` ~ `番剧权限4` | 设置操作权限等级 | 按「番剧权限」 |
| `番剧设置` | 查看本群当前设置 | 按「番剧权限」 |
| `番剧设置 每日上限 数字` | 设置每天最多可添加番剧数（1~20） | 按「番剧权限」 |

**操作权限等级：**

| 等级 | 含义 |
|------|------|
| `1` | 仅 Bot 管理员 |
| `2` | 仅群主 |
| `3` | 群主 + 群管理员（默认） |
| `4` | 全员 |

该权限同时作用于：上传、进度更新、删除、移动、交换、清空、修改设置、修改权限、推送管理。Bot 管理员始终拥有全部权限。

### 定时推送

| 指令 | 说明 | 权限 |
|------|------|------|
| `番剧推送` | 查看本群推送状态与可用子命令 | 所有人 |
| `番剧推送 开启` | 开启本群每日定时推送 | 按「番剧权限」 |
| `番剧推送 关闭` | 关闭本群每日定时推送 | 按「番剧权限」 |
| `番剧推送 时间 8:30` | 设置每日推送时间（24 小时制） | 按「番剧权限」 |
| `番剧推送 立即` | 立即推送今日番剧（测试用） | 按「番剧权限」 |

开启定时推送后，Bot 会在设定时间自动向群内发送今日番剧列表（含进度 + 单行拼接长图），每天仅推送一次。推送文案会提示可用指令与表情回应。

## 权限说明

- **Bot 管理员**：在 AstrBot 主配置的 `admins_id` 中配置的用户，始终可执行全部操作
- **群主 / 群管理员**：由消息平台透传的群身份（如 QQ OneBot 的 `owner` / `admin`）
- 群内可通过 `番剧权限3` 等指令调整本群操作权限
- 旧版 `upload_mode`（0/1/2）会自动迁移为新权限等级
## 数据存储

插件数据保存在：

```
data/plugin_data/astrbot_plugin_anime_schedule/<群号>/
├── schedule.json   # 番剧排期数据
├── settings.yml    # 群内设置
└── *.jpg / *.png   # 番剧封面与生成的周表图
```

## 常见问题

**Q：生成周表长图失败？**

请确认服务器已安装 Pillow：`pip install Pillow`。若未安装，Bot 会提示安装命令。

**Q：上传时提示需要图片？**

`番剧上传` 必须附带图片，或在同一条消息中引用一条带图的消息。仅文字无法添加。

**Q：定时推送没有触发？**

1. 确认已执行 `番剧推送 开启`
2. 确认 Bot 有主动发送群消息的权限
3. 推送时间使用 24 小时制，如 `番剧推送 时间 8:30`
