# 番剧追番表

AstrBot 插件，用于在群内按周一至周日记录追番列表，支持图片 + 文字上传，生成周表长图，以及每日定时推送今日番剧。

## 功能概览

- 按星期（周一至周日）管理番剧，每条记录包含名称与封面图
- 一键生成每周追番长图（`番剧图`）
- 查看今日或指定星期的番剧列表（含封面图）
- 群内可配置上传权限、每日添加上限
- 支持每日定时推送今日番剧到群聊

## 环境要求

- AstrBot `>= 4.16, < 5`
- 支持平台：aiocqhttp、QQ 官方机器人、Telegram

## 安装

1. 将本插件放入 AstrBot 的 `plugins` 目录，或通过插件市场安装
2. 安装依赖：

```bash
pip install -r requirements.txt
```

主要依赖：`Pillow`（生成周表长图）、`aiohttp`（下载图片）

3. 重启 AstrBot 使插件生效

## 插件配置

在 AstrBot 插件配置中可设置以下全局默认值（各群可在群内指令中单独覆盖）：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `upload_mode_default` | 默认上传权限：`0` 关闭，`1` 仅 Bot 管理员，`2` 全体成员 | `2` |
| `max_per_day_default` | 每天默认可添加番剧数量上限（1~20） | `10` |
| `push_hour_default` | 默认定时推送小时（0~23） | `8` |
| `push_minute_default` | 默认定时推送分钟（0~59） | `0` |

## 指令一览

所有指令均在**群聊**中使用。部分指令支持 `/` 前缀（如 `/番剧帮助`）。

### 帮助

| 指令 | 说明 |
|------|------|
| `番剧帮助` | 显示指令帮助（别名：`/番剧帮助`、`番剧菜单`） |

### 上传与查看

| 指令 | 说明 |
|------|------|
| `番剧上传 周X 名称` + 图片 | 添加番剧到指定星期，需附带图片或引用带图消息 |
| `番剧图` | 生成每周追番长图（别名：`/番剧图`、`番剧列表`、`/番剧列表`） |
| `今日番剧` | 查看今天更新的番剧（文字列表 + 单行拼接长图） |
| `番剧 周X` | 查看指定星期的番剧（文字列表 + 单行拼接长图） |

**星期写法支持：** `周一`~`周日`、`周1`~`周7`、`星期一`~`星期日`、`1`~`7` 等。

**上传示例：**

```
番剧上传 周一 葬送的芙莉莲
（同时发送一张封面图，或引用一条带图的消息）
```

### 删除与清空

| 指令 | 说明 | 权限 |
|------|------|------|
| `删除番剧 周X 编号` | 删除某天指定番剧（编号从 1 开始） | Bot 管理员 |
| `清空番剧 周X` | 清空某天的全部番剧 | Bot 管理员 |
| `清空番剧 全部` | 清空本周全部番剧 | Bot 管理员 |

**示例：**

```
删除番剧 周一 1
清空番剧 周三
清空番剧 全部
```

### 权限与设置

| 指令 | 说明 | 权限 |
|------|------|------|
| `番剧权限` | 查看当前上传权限 | 所有人 |
| `番剧权限0` / `番剧权限1` / `番剧权限2` | 设置上传权限 | Bot 管理员 |
| `番剧设置` | 查看本群当前设置 | Bot 管理员 |
| `番剧设置 每日上限 数字` | 设置每天最多可添加番剧数（1~20） | Bot 管理员 |

**上传权限说明：**

- `0`：关闭上传
- `1`：仅 Bot 管理员可上传
- `2`：全体成员可上传

### 定时推送

| 指令 | 说明 | 权限 |
|------|------|------|
| `番剧推送` | 查看本群推送状态与可用子命令 | 所有人 |
| `番剧推送 开启` | 开启本群每日定时推送 | Bot 管理员 |
| `番剧推送 关闭` | 关闭本群每日定时推送 | Bot 管理员 |
| `番剧推送 时间 8:30` | 设置每日推送时间（24 小时制） | Bot 管理员 |
| `番剧推送 立即` | 立即推送今日番剧（测试用） | Bot 管理员 |

开启定时推送后，Bot 会在设定时间自动向群内发送今日番剧列表（文字 + 单行拼接长图），每天仅推送一次。

## 权限说明

- **Bot 管理员**：在 AstrBot 主配置的 `admins_id` 中配置的用户
- 删除、清空、权限设置、推送设置等管理操作仅限 Bot 管理员
- 上传权限可通过 `番剧权限` 在群内单独控制

## 数据存储

插件数据保存在：

```
data/plugin_data/astrbot_plugin_anime_schedule/<群号>/
├── schedule.json   # 番剧排期数据
├── settings.yml    # 群内设置
└── *.jpg / *.png   # 番剧封面与生成的周表图
```

## 常见问题

**Q：生成周表长图失败？**

请确认服务器已安装 Pillow：`pip install Pillow`。若未安装，Bot 会提示安装命令。

**Q：上传时提示需要图片？**

`番剧上传` 必须附带图片，或在同一条消息中引用一条带图的消息。仅文字无法添加。

**Q：定时推送没有触发？**

1. 确认已执行 `番剧推送 开启`
2. 确认 Bot 有主动发送群消息的权限
3. 推送时间使用 24 小时制，如 `番剧推送 时间 8:30`
